parseAnsysNodeLine() → ansysCoordinate object
Each data line: nodeID X Y Z (mm, ANSYS global CS).
Strips leading/extra spaces via regex, splits on whitespace.
Stores as ansysCoordinate class: {inode, x, y, z}.

Phase 3 — Geometry: Coordinate System Definition
📐
sourceGeometry.initialized() — Build orthonormal basis
Three control points read from form1:
 · Source (xₛ,yₛ,zₛ) — photon source point (e.g. centre of undulator)
 · Target (xₜ,yₜ,zₜ) — beam axis reference point (defines ê_Z)
 · Side (xₚ,yₚ,zₚ) — in-plane side point (defines ê_X direction)

Constructs local basis:
  ê_Z = (Target − Source) / |…|
  ê_Y = ê_Z × ê_X_raw  (cross product)
  All three vectors orthonormalized via orthonormalize().
Displayed on form as Base vector X/Y/Z.

Step 4b — mapAnsysCoordinateToMrad() — Project to angular space
Displacement from source: v = (x̄−xₛ, ȳ−yₛ, z̄−zₛ)
Distance: R = |v| [mm]
Angular projection using local basis vectors:
xmrad = arcsin( v · ê_X / R ) × 1000
ymrad = arcsin( v · ê_Y / R ) × 1000
Note: arcsin valid for small-angle regime (<5 mrad), appropriate for synchrotron undulator radiation.

📊
Step 4c — Bilinear interpolation of power density
Retrieve pre-computed coefficients a₀…a₃ from spectraElement.interpolateCoefficients.
φ_normal = a₀ + a₁·xmrad + a₂·ymrad + a₃·xmrad·ymrad  [kW/mrad²]

Coefficient derivation (Wikipedia bilinear, node layout as above):
a₀ = (f₂·x₂y₂ − f₁·x₂y₁ − f₄·x₁y₂ + f₃·x₁y₁) / Δ
a₁ = (−f₂·y₂ + f₁·y₁ + f₄·y₂ − f₃·y₁) / Δ
a₂ = (−f₂·x₂ + f₁·x₂ + f₄·x₁ − f₃·x₁) / Δ
a₃ = (f₂ − f₁ − f₄ + f₃) / Δ   where Δ = (x₂−x₁)(y₂−y₁)
powerDensityElement
↓
📐
Step 4d — getGrazingAngleInRadian() — Surface normal & beam incidence
Edge vectors from corner node 2 (pivot):
v₁ = node1 − node2  (ji direction)
v₂ = node3 − node2  (jk direction)
Surface normal: n = v₂ × v₁ (normalized)  ← code reverses order vs comment; Abs() compensates.
Grazing angle: θ = |π/2 − arccos( n̂ · v̂ )|
where v̂ = (x̄−xₛ, ȳ−yₛ, z̄−zₛ) / R is the beam direction unit vector.
ansysHeatFlux
↓
⚗️
Step 4e — Unit conversion and projected power density
Angular → spatial flux (inverse-square law):
φ_normal [W/mm²] = φ_normal [kW/mrad²] × 10⁹ / R[mm]²
    (= × 1000 / (R/1000)²)
Projection onto surface (Lambert's cosine for grazing geometry):
φ_projected [W/mm²] = sin(θ) × φ_normal [W/mm²]
Total power on element:
P_total [W] = A_surface [mm²] × φ_projected [W/mm²]

Phase 5 — Output: Write ANSYS External Data File
💾
writeAnsysHeatFluxElements() — Generate .inp file
Reads totalPowerRatio from form (scaling factor for partial-beam scenarios, default 1.0).
For each heat flux element, writes one CSV line:
x, y, z, φ_proj×ratio, φ_proj [0.0000E+00 format]
Known issue: arrayString(10) declaration adds 6 trailing zero columns — see Bug #1 below.
Updates form labels: total elements output, total power in kW.
Enables View file location button. Shows completion MsgBox.
On exit: backupFormParameters() saves all settings to .backup.txt in ANSYS file directory.
